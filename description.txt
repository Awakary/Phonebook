Телефонный справочник, выводящий записи в консоль

Скрипт реализован в файле main, в который импортируются классы одной
записи (PhoneNote) и страницы справочника (Page), также в него импортируются
функции валидации введенных данных.
В main по введенной цифре от пользователя (от 1 до 5) выбирается нужная команда:
- постраничный вывод справочника(нужная страница или все)
- поиск записей по одному или нескольким критериям
- добавление записей
- редактирование записей

Create_phonebook.py создает справочник записей с помощью модуля random создаются
записи справочника. После создания справочник записывается в json-файл.

Модуль validation содержит две функции валидации:
- validation_int - проверяет корректность введеных чисел и конвертирует их в int
- validation_field - проверяет корректность введенных данных для вносимых записей справочника

Модуль classes включает в себя  два класса:
1) класс PhoneNote - для одной записи справочника, атрибуты которого являются полями
записи(фамилия, имя и т.д.). Класс PhoneNote имеет следующие методы:
- статический метод search_note ищет записи по одному или нескольким критериям
- статический метод writing_sorted_phonebook - записывает отсортированные записи
справочника в файл json
- метод print_note печатает запись
2) класс Page - для формирования страницы справочника, атрибуты которого - определенное
количество записей и номер страницы
класс Page имеет метод print_notes_from_page - печатает все записи со страницы с номером страницы
