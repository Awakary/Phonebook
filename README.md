
Phonebook Телефонный справочник, реализованный на Python, в json-файле

Справочник хранит информацию: ФИО, название организации, рабочий телефон, мобильный телефон

Сам справочник хранится в json-файле Phonebook.json, загружается из него и отредактированный записывается в него Реализация интерфейса осуществляется через консоль с помощью выбора команды пользователем (цифры от 1 до 5)

Справочник создан с помощью модуля create_phonebook.py, в котором из списков с данными формируется рандомный список словарей и записывается в json-файл

Работа со справочником осуществляется с помощью модуля __main__.py (функция choose_operation() - выбор действия пользователем, функция main)

В модуле print_functions: функция print_phone()- печать одной записи, функция print_pages - постраничный вывод справочника целиком или необходимой страницы по номеру

В модуле add - функция add_note() - добавление записи со всеми проверками на правильность введенных данных

В модуле redakt - функция redakt_note() - редактирование записи

В модуле  search - функция search_note()- поиск записей по справочнику по одному или нескольким параметрам
