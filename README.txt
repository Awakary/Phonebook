Phonebook Телефонный справочник, реализованный на Python, в json-файле

Справочник хранит информацию: ФИО, название организации, рабочий телефон, мобильный телефон

Сам справочник хранится в json-файле Spravochnik.json, загружается из него и отредактированный записывается в него Реализация интерфейса осуществляется через консоль с помощью выбора команды пользователем (цифры от 1 до 5)

Справочник создан с помощью файла Sozdanie_Spravochnika.py, в котором из списков с данными формируется рандомный список словарей и записывается в json-файл

Работа со справочником возможна с помощью файла vubor_deistviya.py - в нем реаилизована основная программа и созданы все функции по работе со справочником:

функция vubor() - выбор действия пользователем

функция pechat()- вывод постранично справочника целиком или необходимой страницы по номеру

функция add_zapis() - добавление записи со всеми проверками на правильность введенных данных

функция redakt() - редактирование записи

функция poisk()- поиск записей по справочнику по одному или нескольким параметрам
